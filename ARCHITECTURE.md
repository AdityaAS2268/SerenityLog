# SerenityLog – System Architecture

## Overview

SerenityLog is an AI-assisted journaling system that analyzes user emotions from journal entries and generates insights using a Large Language Model.

The system consists of:

Frontend:

- React (Vite)

Backend:

- Node.js
- Express.js

AI Layer:

- Google Gemini API

Database:

- SQLite

Caching / Rate Control:

- Redis
- Express rate limiter
